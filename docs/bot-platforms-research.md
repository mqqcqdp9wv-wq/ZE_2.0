# Платформы-конструкторы VK-ботов — Исследование

Обзор платформ для построения чат-ботов ВКонтакте с визуальными конструкторами, шаблонами воронок и примерами для сервисного бизнеса.

---

## 1. Senler (senler.ru)

**Специализация:** VK-рассылки и чат-боты. Самая популярная платформа для ВКонтакте.

### Возможности
- Визуальный конструктор ботов (drag-and-drop)
- Рассылки подписчикам через ВК
- Карусели, inline-клавиатуры, кнопки
- Автоворонки продаж
- Сегментация подписчиков (теги, переменные)
- Таймеры, отложенные сообщения
- Интеграция с CRM, webhook
- Статистика и аналитика

### Шаблоны воронок
- Приветственная цепочка (welcome → знакомство → оффер)
- Лид-магнит (подписка → полезный контент → продажа)
- Квиз-воронка (вопросы → сегментация → персональный оффер)
- Прогрев → продажа (серия сообщений с контентом)

### Как устроен визуальный конструктор
- Узлы (ноды) на холсте — сообщения, условия, действия
- Стрелки-связи между узлами (переходы)
- Каждый узел: текст + медиа + кнопки + задержка
- Условные ветвления (по тегам, переменным, действиям)
- Предпросмотр сообщений

### Для ze.studio (автотонирование)
- Приветствие → меню услуг
- Квиз: «Какую плёнку выбрать?» (тип авто → цели → рекомендация)
- FAQ-бот с кнопочным меню
- Каталог плёнок через карусель
- Запись на тонирование через бота

### Цены
- **Бесплатно:** до 50 подписчиков, базовый функционал бота
- **Старт:** ~219 ₽/мес — до 200 подписчиков
- **Бизнес:** ~569 ₽/мес — до 2000 подписчиков
- **Про:** от ~1500 ₽/мес — неограниченно

### Экспорт
- Нет прямого экспорта в JSON
- Есть API для управления подписчиками и рассылками
- Структуру бота можно описать и воспроизвести вручную

## 2. BotHelp (bothelp.io)

**Специализация:** мультиплатформенные чат-боты (VK, Telegram, WhatsApp, Instagram).

### Возможности
- Визуальный конструктор воронок
- Мультиканальность (один бот → несколько платформ)
- Карусели, кнопки, быстрые ответы
- CRM с карточками клиентов
- Теги, сегментация, переменные
- Broadcast-рассылки
- A/B тестирование сообщений
- Webhook и интеграции (Zapier, HTTP)

### Шаблоны воронок
- Welcome-воронка (приветствие → знакомство → квалификация)
- Продуктовый каталог (меню → карточки → детали → заявка)
- FAQ-бот (меню вопросов → ответы → возврат)
- Запись на услугу (выбор услуги → дата → подтверждение)
- Обратная связь (оценка → комментарий → благодарность)

### Структура потока в BotHelp
```
Trigger (сообщение/подписка)
  → Step 1: Сообщение + кнопки
     → Branch A: Кнопка "Услуги"
        → Step 2A: Каталог услуг
     → Branch B: Кнопка "FAQ"
        → Step 2B: Список вопросов
     → Branch C: Кнопка "Контакт"
        → Step 2C: Форма сбора данных
```

### Для ze.studio
- Можно собрать прототип воронки визуально
- Протестировать UX карусели и кнопок в ВК
- Затем перенести логику в свой бот

### Цены
- **Бесплатно:** тестовый период 14 дней
- **Starter:** ~$15/мес (≈1500 ₽)
- **Pro:** ~$30/мес (≈3000 ₽)
- Оплата в долларах (иностранный биллинг)

### Экспорт
- Нет экспорта структуры бота
- API для управления контактами и рассылками

## 3. SaleBot (salebot.pro)

**Специализация:** CRM + чат-боты для продаж. Российский сервис.

### Возможности
- Визуальный конструктор (типа блок-схемы)
- ВК, Telegram, WhatsApp, Instagram, Viber, Avito
- Встроенная CRM с воронкой продаж
- Калькуляторы, формы, опросы
- Переменные, условия, вычисления
- Интеграция с платёжными системами (ЮKassa, Тинькофф)
- Webhook, HTTP-запросы

### Шаблоны
- CRM-воронка (лид → квалификация → оффер → сделка)
- Каталог товаров/услуг
- Квиз-бот
- Бот для записи
- Рассыльщик

### Для ze.studio
- Полноценная CRM + бот в одном
- Можно вести заявки, статусы, историю клиентов
- Каталог плёнок + запись + ведение сделки

### Цены
- **Бесплатно:** до 3 проектов, базовый функционал
- **Стартап:** ~699 ₽/мес
- **Бизнес:** ~1499 ₽/мес
- **Все тарифы в рублях** (российский биллинг)

### Экспорт
- Нет экспорта в JSON/YAML
- API для интеграций

## 4. Другие платформы

### Chatgun (chatgun.io)
- Специализируется именно на ВК
- Рассылки, автоответы, простой бот
- Дешёвый: от 390 ₽/мес
- Без визуального конструктора воронок — более простой

### Robochat (robochat.io)
- Простой конструктор ботов для ВК
- Правила «если-то»: ключевые слова → ответы
- Подходит для простых FAQ-ботов
- Бесплатный базовый тариф
- Нет карусели и сложных меню

### TextBack
- Мультиканальный чат-центр
- ВК, Telegram, WhatsApp, SMS
- Больше для поддержки, чем для воронок
- Ребрендинг → часть BotHelp

### ChatPlace
- Автоворонки для ВК и Instagram
- Визуальный редактор
- От 990 ₽/мес

## 6. Best Practices для VK-бота автосервиса

### Структура приветственной воронки

```
Пользователь пишет в группу
  ↓
[Приветствие] — краткое, тёплое, с логотипом
  ↓
[Главное меню] — 4-5 кнопок
  ├── Наши плёнки → [Карусель: 4-6 карточек с фото]
  │     └── Подробнее → [Текст + кнопка "Узнать цену"]
  ├── Услуги и цены → [Описание + сроки работ]
  ├── О студии → [Опыт, награды, сертификаты]
  ├── Частые вопросы → [Кнопочное FAQ-меню]
  │     ├── Почему дороже? → [Ответ + Назад]
  │     ├── Сколько времени? → [Ответ + Назад]
  │     └── ...
  └── Записаться → [Контакты / форма]
```

### Что работает для автосервисов

1. **Карусель продуктов** — самый эффектный способ показать плёнки. Фото на реальных авто + краткие характеристики
2. **Квиз «Подберём плёнку»** — 3-4 вопроса (тип авто, цели, бюджет) → персональная рекомендация
3. **До/После** — галерея работ вызывает максимальное доверие
4. **Быстрый контакт** — кнопка «Позвонить» или «WhatsApp» всегда на виду
5. **Социальное доказательство** — ссылка на отзывы Яндекс.Карт в приветствии

### Ошибки, которых стоит избегать

- Слишком длинные тексты в боте (максимум 3-4 абзаца на экран)
- Слишком глубокая вложенность (не больше 3 уровней)
- Нет кнопки «Назад» или «В меню»
- Карусель без изображений (теряет 80% эффекта)
- Принуждение к подписке до показа контента

### Метрики для отслеживания

- Количество уникальных пользователей, начавших диалог
- Процент прохождения воронки (welcome → contact)
- Самые популярные разделы (по кликам)
- Конверсия в заявку/звонок
- Время от первого сообщения до заявки

## 7. Рекомендация для ze.studio

### Вариант A: Прототип в Senler → Свой бот

1. Зарегистрироваться в Senler (бесплатно до 50 подписчиков)
2. Собрать воронку визуально — протестировать UX
3. Зафиксировать структуру, тексты, изображения
4. Перенести в свой бот (`api/vk.js`) с JSON-структурой из `vk-bot-menu-structure.json`

**Плюсы:** быстрый прототип, визуальное проектирование
**Минусы:** нужно переносить руками

### Вариант B: Сразу свой бот с JSON-конфигом

1. Доработать `api/vk.js` — добавить движок чтения JSON-структуры
2. Добавить обработку `message_event` (callback-кнопки)
3. Загрузить изображения через VK API
4. Тестировать в тестовой группе

**Плюсы:** полный контроль, бесплатно, интеграция с Telegram
**Минусы:** нет визуального редактора (но можно сделать позже)

### Вариант C: SaleBot как основа

1. Использовать SaleBot для бота + CRM
2. Вести заявки и клиентов в их системе
3. Webhook для отправки уведомлений в Telegram

**Плюсы:** CRM из коробки, визуальный конструктор
**Минусы:** платная подписка, зависимость от сервиса, меньше контроля

### Наша рекомендация: Вариант A → B

Начать с прототипа в Senler (бесплатно), отточить UX, потом перенести в свой бот с полным контролем. JSON-структура (`vk-bot-menu-structure.json`) уже готова к использованию.
